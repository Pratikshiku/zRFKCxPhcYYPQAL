## Online Cycling Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1411709be41f4ccfb7891bd9942f606b/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Forum Information
4. Route Guides
5. News Events
6. Cycling Sharing
7. Beginner Knowledge
8. Personal Center
9. Route Guide Favorites
10. Cycling Sharing Favorites
11. Beginner Knowledge Favorites


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Route Type Management, News Type Management, Cycling Record Type Management, Knowledge Type Management)
5. Forum Information Management
6. Route Guide Management
7. Route Guide Comment Management
8. Route Guide Favorite Management
9. News Event Management
10. Cycling Record Sharing Management
11. Cycling Record Sharing Comment Management
12. Cycling Record Sharing Favorite Management
13. Beginner Knowledge Management
14. Beginner Knowledge Comment Management
15. Beginner Knowledge Favorite Management
16. User Management
17. Carousel Management
